# Children in Forest🎄
--------------
## Project Purpose
> 유아용 게임 앱 개발을 목적으로 하고있다. 주 소비자층이 유아들이므로 정서에 맞게 아기자기하며 동화적인 분위기를 이끌어냈으며 안드로이드 플랫폼으로 제작하였다. 하지만 유아의 스마트폰의 잦은 사용으로 인한 부정적인 시각도 일부 존재하는데, 이로 인해 언어, 정서, 사회성 발달 장애를 불러온다는 것이다. 우리는 이러한 문제점을 인지하였으며 단순한 오락 용도가 아닌 유아의 올바른 성장을 위한 상식, 언어교육 등을 게임과 접목시켜 본 앱을 통해 아이가 언어적 정신적 성장을 할 수 있게 하였다.
-----------
## Project Preview
> ### **Intro**
>
> 회사의 로고가 페이드 인 아웃으로 화면에 나타났다가 사라지며, 다음 화면인 메인 화면이 나오게 된다.
>
> > <img src = "https://user-images.githubusercontent.com/69896751/126109396-99a7437e-1cf6-42a0-88c6-11fd040352b8.png" width="500" height="220">
> ### **Intro main**
>
> 아이들이 좋아할 만한 테마의 메인 화면이 등장한다. 이때 스테이지 선택 화면으로 넘어가기 위한 Start button이 존재하게 된다.
>
> > <img src = "https://user-images.githubusercontent.com/69896751/126109764-dafe0986-6b18-4aaa-95f1-3b0984a5253a.png" width="500" height="220">
> ### **Stage select**
>
> Start button이 눌리면, Stage select 화면으로 넘어오게 된다. 여기서 각각의 미니 게임 스테이지 아이콘을 클릭하면 들어갈 수 있게 된다.
>
> > <img src = "https://user-images.githubusercontent.com/69896751/126110335-8d81128c-2fd1-4770-8422-b682df6c2c39.png" width="500" height="220">
> ### **Game stage**
>
> Stage select 화면에서 미니 게임 스테이지 아이콘을 클릭하면, 각각의 미니 게임 스테이지로 넘어간다. 이때 로딩 화면 또한 구성할 예정이다.
>
> **1. 충치 없애기 게임**
>
> 충치를 없애는 것을 해보면서 이를 닦지 않으면, 게임에서 나오는 충치균들이 발생한다는 것을 아이에게 연상시켜 주는 것을 목적으로 한 게임이다.화면 구성으로는 아이와 아이의 이를 보여준다. 이때 충치균들이 이에 발생하는데 이를 단순히 클릭으로 없애는 게임이다. 일정 수의 충치균을 없애면 게임을 마치게 된다. 향후 추가할 내용으로, 충치균들이 일정 수 만큼 늘어나게 되면 게임오버 시켜 아이에게 이를 닦지 않았을 때 어떻게 되는 지를 보여준다.
>
> > <img src = "https://user-images.githubusercontent.com/69896751/126112735-c97754b3-ac49-4416-a178-38e1161ccce1.png" width="500" height="220">
> **2. 퍼즐 게임**
>
> 아이의 공간지각력 향상을 위한 단순한 퍼즐 게임이다. 화면 구성으로는 아이들이 좋아하는 ip인 공룡을 활용하여 친숙한 퍼즐 게임을 구성하였다. 9조각의 퍼즐조각으로 구성되어 있으며, 오른쪽에 임의로 배치된 퍼즐 조각을 드래그해서 왼쪽의 퍼즐판에 맞추는 게임이다. 퍼즐을 완성시키면 게임을 마치게 된다. 향후 퍼즐의 종류를 늘려 퍼즐을 맞추었을 때, 공룡이면 한국어로 공룡, 영어로 공룡의 이름을 말로써 들려주어 연상시키도록 할 예정이다.
>
> > <img src = "https://user-images.githubusercontent.com/69896751/126113341-32ddbf54-77a5-4aa1-a5cd-7f439fb1ab06.png" width="500" height="220">
> **3. 과일 넣기 게임**
>
> 아이의 언어 발달을 위한 게임으로써, 화면 구성으로는 과일을 담을 수 있는 과일 바구니와 임의의 위치에 생성되는 과일들로 구성된다. 과일의 종류로는 바나나, 사과, 파인애플, 포도, 딸기가 있다. 화면 상의 과일들을 바구니에 드래그하여 넣으면, 게임을 마치게 된다. tts를 이용하여 과일의 이름을 말로써 아이에게 들려주어 과일의 이름을 연상시킬 수 있도록 하였다.
>
> > <img src = "https://user-images.githubusercontent.com/69896751/126114122-93dd17d5-abb3-4f2f-aa98-6c48316fa167.png" width="500" height="220">
> **4. 도형 맞추기 게임**
>
> 아이의 공간지각력 향상을 위한 단순한 도형 맞추기 게임이다. 화면 구성으로는 왼쪽에는 도형 틀과 오른쪽에는 도형이 존재한다. 이때 플레이어는 도형을 드래그하여 맞는 도형틀에 집어 넣어야 한다. 향후 틀리면 틀렸다는 효과음과 애니메이션을 추가할 예정이다.
>
> > <img src = "https://user-images.githubusercontent.com/69896751/126114831-52595f15-3f57-462a-957e-0c3890d686e7.png" width="500" height="220">
> **5. 마트 장보기 게임**
>
> 아이의 인지능력 향상을 위한 마트 장보기 게임이다. 화면 구성으로는 캐릭터와 카트, 캐릭터가 생각하는 물건이 뜨는 말풍선이 있다. 말풍선에 뜨는 물건을 카트에 담으면 되는 간단한 게임이다.
>
> > <img src = "https://user-images.githubusercontent.com/69896751/126115882-cc6d8471-5213-4b04-a683-68d7da05d190.png" width="500" height="220">
> **6. 동화 게임**
>
> 동화를 게임 어플에 녹여놓은 형태로써, e북 처럼 보호자가 아이에게 읽어주는 게임이다. 아이 혼자서 진행해도 되지만, 보호자가 동화 게임을 진행하는 것을 도와주어(읽어 주거나, 게임 진행에 도움을 주는 역할) 게임에 대한 보호자의 긍정적인 반응을 이끌어 내기 위한 게임이다.
>
> > <img src = "https://user-images.githubusercontent.com/69896751/126115996-6e49ada7-41bc-4419-8f80-c2fd5bb560ed.png" width="500" height="220">

> ### **End scene**
>
> 각각의 게임 스테이지를 마치면, 아기자기하고 귀여운 테마의 화면이 잠깐 나오고 다시 Stage select 화면으로 넘어가게 된다.
